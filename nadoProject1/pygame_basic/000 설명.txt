프레임 속도와 이동 속도의 관계
    프레임(Frame)**은 동영상, 애니메이션, 게임 등에서 사용되는 개별 이미지
    FPS(Frame Per Second)**는 초당 표시되는 프레임의 수
    1. 프레임 기반 이동의 개념
    프레임 기반 이동은 게임 루프의 각 프레임마다 캐릭터의 위치를 일정한 값만큼 이동시키는 방식입니다. 예를 들어, 매 프레임마다 캐릭터의 x 좌표를 5씩 증가시킨다면:
    character_x_pos += 5  # 매 프레임마다 5픽셀 이동
    2. 프레임 속도에 따른 이동 속도 변화
    프레임 속도가 높을수록 (즉, FPS가 높을수록) 루프가 더 자주 실행되기 때문에, 캐릭터가 더 자주 5픽셀을 이동하게 됩니다. 이는 결국 초당 이동하는 픽셀 수가 증가하게 됩니다.

    예시:
    FPS = 30일 때:
    초당 이동 픽셀: 5픽셀 × 30프레임 = 150픽셀

    FPS = 60일 때:
    초당 이동 픽셀: 5픽셀 × 60프레임 = 300픽셀
    즉, 프레임 속도가 두 배로 증가하면 초당 캐릭터의 이동 거리도 두 배로 증가하게 됩니다.

해결 방법: 델타 타임(dt) 사용하기
    character_speed : 초당 이동할 픽셀수
    dt : 프레임당 시간
    charater_speed * dt = 한 프레임당 이동 픽셀수
예시
    FPS = 30일 때:
    dt_seconds ≈ 0.033초
    이동 거리: 200 * 0.033 ≈ 6.6픽셀

    FPS = 60일 때:
    dt_seconds ≈ 0.016초
    이동 거리: 200 * 0.016 ≈ 3.2픽셀
    두 경우 모두 초당 이동 거리는 약 200픽셀로 유지됩니다.
